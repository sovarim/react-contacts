# React contacts

## Установка
Выполните команду
```shell
yarn install 
```
Запустите сервер
```shell
yarn start:server
```
Запустите клиент
```shell
yarn start
```

## Действия после запуска
В приложении необходимо авторизоваться, можете использовать следующие данные для этого:
```
login: test
password: testpassword
```
или добавить нового пользователя в файл `db.json` в массив `users`
